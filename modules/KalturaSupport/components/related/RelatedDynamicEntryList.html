Related Plugin: Dynamic entryList

You can dynamically change the entryList property of the Related plugin thus changing the related entries shown in the related screen.
Click the "Change Entries" button below the player to change the related entries.





Code sample: 
kdp.setKDPAttribute('related', 'entryList', '0_33vkwid6,1_18leun9q,1_23pqn2nu');
